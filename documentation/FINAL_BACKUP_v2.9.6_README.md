## 🎯 THEME-SYSTEM

### Automatische Theme-Zuordnung

Jeder Chat-Raum hat automatisch ein passendes Theme:

```dart
// In lib/models/chat_room_model.dart
String get backgroundTheme {
  final themeMap = {
    'room_mystery': 'mystery',       // 🔮 Mysterien & Rätsel
    'room_wisdom': 'wisdom',         // 📚 Weisheit & Philosophie
    'room_alchemy': 'alchemy',       // ⚗️ Alchemie & Transformation
    'room_cosmos': 'cosmos',         // 🌌 Kosmos & Sterne
    'room_nature': 'nature',         // 🌿 Natur & Elemente
    'room_ancient': 'ancient',       // 🏛️ Antike Zivilisationen
    'room_energy': 'energy',         // ⚡ Energie & Frequenzen
    'room_art': 'art',               // 🎨 Kunst & Kreativität
    'general': 'library',            // 📚 Allgemeiner Chat
    'music': 'music',                // 🎵 Musik-Chat
  };
  return themeMap[id] ?? 'library';
}
```

### Verwendung im Code

```dart
// In lib/screens/chat_room_detail_screen.dart
body: ThemedChatBackground(
  theme: widget.chatRoom.backgroundTheme,  // Automatisch!
  child: Stack([
    // Dein Chat-Inhalt
  ]),
)
```

### Theme-Struktur

Jedes Theme besteht aus:
1. **4-Farben-Gradient** (LinearGradient)
2. **CustomPainter Muster** (Sterne, Kreise, Wellen, etc.)
3. **Dunkles Overlay** (für Lesbarkeit)
