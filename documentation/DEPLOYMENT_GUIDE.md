2. Zahle die einmalige Registrierungsgebühr ($25 USD)

## 🔐 Content Rating

Google Play erfordert ein Content Rating (IARC-System).

**Für Weltenbibliothek:**

**Altersempfehlung**: 16+ oder 18+  
**Grund**: Kontroverse Themen, Verschwörungstheorien, möglicherweise verstörende Inhalte

**Content Rating Fragebogen:**
- Gewalt: Keine/Minimal
- Sexualität: Keine
- Sprache: Moderat (Community-Chat)
- Drogen/Alkohol: Thematisch relevant (Historie)
- Glücksspiel: Nein
- Nutzergenerierte Inhalte: Ja (Chat, DM)
- Standortdaten: Ja (Map-Feature)
- Persönliche Daten: Minimal (lokale Speicherung)

**Content Rating beantragen:**
1. In Play Console → "Content Rating"
2. Fragebogen ausfüllen (ca. 10 Minuten)
3. E-Mail-Verifizierung
4. Rating wird automatisch vergeben

## ⚠️ Wichtige Hinweise

### Backend-Server Konfiguration

**Problem**: yt-dlp Backend läuft auf `localhost:5001`

**Lösung für Production:**

**Option 1: Cloud-Hosting (empfohlen)**
- Hoste `ytdlp_api_server_v2.py` auf Cloud-Provider (Heroku, AWS, DigitalOcean)
- Ersetze `localhost:5001` in Flutter-Code mit öffentlicher URL
- Beispiel: `https://api.weltenbibliothek.com`

**Option 2: User-eigener Server**
- User startet Backend auf eigenem Gerät
- App verwendet lokales Netzwerk (WiFi)
- **Nachteil**: Nur im lokalen Netzwerk funktionsfähig

**Code-Anpassung erforderlich:**
```dart
// lib/services/ytdlp_api_service.dart
class YtdlpApiService {
  // ÄNDERN VON:
  static const String baseUrl = 'http://localhost:5001/api/v1';
  
  // ÄNDERN ZU:
  static const String baseUrl = 'https://api.weltenbibliothek.com/api/v1';
}
```

**Rebuild nach Änderung:**
```bash
cd /home/user/weltenbibliothek
flutter clean
flutter build apk --release
```
